### Atomik Portal de Clientes
    
Esta aplicación permite gestionar tu información como cliente de Atomik.

## Tecnologías

- Next.js
- React
- TypeScript
- TailwindCSSI
- NextAuth.js
- Husky
- ESLint
- Prettier
- Driver.js
- PostgreSQL
